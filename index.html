提升开发效率的 VS Code 配置技巧

众所周知，VS Code 是一款功能强大的代码编辑器，通过合理的配置可以显著提升开发效率。以下是一些实用的配置技巧：

1. 安装必备扩展

根据你的开发需求，安装一些常用的扩展。

例如，如果你喜欢在VS Code里写GitHub仓库的README.md文件，可以安装Markdown Preview Enhanced扩展，它提供了更丰富的Markdown预览功能。

如果你喜欢玩C++，可以安装C/C++扩展，它提供了智能代码补全、调试支持和代码导航功能。

2. 配置调试环境

VS Code支持多种编程语言的调试功能。你可以通过配置launch.json文件来设置调试环境。例如，对于C++项目，你可以配置GDB或LLDB调试器，设置断点、变量监视和调用堆栈等功能。

3. 使用VibeCoding配置

VibeCoding是一种更适合小白的编程方式，把你的意念（Vibe）给AI智能体，让它帮你写代码。

VS Code可以使用GitHub Copilot扩展来实现VibeCoding功能。

登陆你的GitHub账户，再进行配置即可使用VibeCoding功能。

每个月还可以得到一些免费的额度（AI对话+内联建议），对于轻度开发来说够用。

如果你是重度开发者，建议购买Copilot Pro，价格为每月10美元，提供无限制的AI建议和对话功能。
